Weather Widget Examples

This page demonstrates how to use the weather widget on any website, regardless of the framework used.

Basic Usage

<weather-widget></weather-widget>

With Custom Options

<weather-widget 
  api-key="YOUR_OPENWEATHERMAP_API_KEY"
  city="London"
  units="metric"
  theme="dark"
></weather-widget>
